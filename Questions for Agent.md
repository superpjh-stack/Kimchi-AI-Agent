# Questions for Agent — 김치 AI Agent 질문 모음

> 각 Phase에서 구현된 기능을 기반으로, 실제로 에이전트에게 할 수 있는 질문들을 정리한 목록입니다.
> 기획서(Plan) + 설계서(Design) 문서 전체를 분석하여 도출했습니다.

---

## Phase 1 — MVP: Chat + RAG + 문서 업로드

> **핵심 기능**: 텍스트/음성 채팅, RAG 기반 문서 질의응답, Quick Questions, 대화 히스토리

### 🌡️ 공정 기본 질문 (Quick Questions)
- 현재 발효실 온도는?
- 오늘 염도 측정 결과는?
- 현재 배치 발효 완료까지 남은 시간?
- 오늘 품질 체크 항목 알려줘
- 이번 주 생산 계획은?
- 현재 이상 경보 있어?

### 📄 문서 기반 RAG 질문
- 김치 발효 최적 온도 기준이 뭐야?
- 배추 절임 시 적정 염도는 얼마야?
- HACCP 절차에서 온도 관리 기준이 뭐야?
- 고춧가루 첨가량 기준 알려줘
- 젓갈 종류별 사용 비율이 어떻게 돼?
- 발효 완료 판단 기준이 뭐야?
- 품질 등급 기준표 보여줘
- 원재료 입고 검사 절차가 어떻게 돼?

### 💬 채팅 기능 관련 질문
- 음성으로 질문할 수 있어?
- 이전에 한 질문 다시 볼 수 있어?
- 대화를 저장할 수 있어?
- 한국어 말고 영어로도 질문할 수 있어?
- 내가 업로드한 문서를 기반으로 답변해줄 수 있어?
- 답변 출처가 어떤 문서야?

### 📁 문서 업로드 관련 질문
- 어떤 파일 형식을 업로드할 수 있어?
- PDF 파일도 업로드 가능해?
- 엑셀 파일로 된 공정 데이터도 학습시킬 수 있어?
- 업로드한 문서가 잘 처리됐어?
- 지금 몇 개의 문서가 등록돼 있어?

## Phase 2 — 공정 데이터 연동 + 영구 저장소

> **핵심 기능**: 실시간 센서 데이터 연동, 이상 감지 알림, 문서 관리 API, 대화 영구 저장

### 🔬 실시간 센서 데이터 질문
- 지금 발효실 온도가 몇 도야?
- 현재 습도 수치는?
- 지금 염도 측정값 알려줘
- 현재 pH는 얼마야?
- 현재 배치 ID가 뭐야?
- 발효 시작한 지 몇 시간 됐어?
- 발효 완료까지 몇 시간 남았어?
- 지난 24시간 온도 변화 추이 보여줘
- 오늘 오전 습도 이력 데이터 알려줘

### ⚠️ 이상 감지 및 알림 질문
- 지금 이상 알림 있어?
- 온도가 임계값을 벗어났어?
- 오늘 경고 알림이 몇 건 발생했어?
- 위험(critical) 이상이 발생했어?
- 알림 설정 온도 범위가 어떻게 돼?
- 염도가 허용 범위 안에 있어?
- pH 이상 징후 감지된 게 있어?

### 📋 문서 관리 질문
- 등록된 문서 목록 보여줘
- 각 문서에서 만들어진 청크가 몇 개야?
- 발효 가이드 문서 삭제해줘
- 전체 벡터 스토어 크기가 얼마야?
- 파일 타입별로 문서 몇 개씩 있어?
- "발효 온도"로 RAG 검색하면 어떤 문서가 나와?

### 💾 영구 저장소 관련 질문
- 이전 대화 기록이 남아 있어?
- 서버 재시작 후에도 문서가 유지돼?
- 지난주 대화를 찾을 수 있어?
- 데이터베이스에 몇 개의 대화가 저장돼 있어?

## Phase 3 — ML 모델 통합 + 대시보드

> **핵심 기능**: 발효 완성도 예측, 품질 등급 분류, 대시보드 탭, 로컬 임베딩(Ollama)

### 🤖 ML 예측 질문
- 현재 배치 발효 완성도가 몇 %야?
- 지금 상태에서 발효가 언제 완료될 것 같아?
- 이 배치 품질이 A/B/C 중 어느 등급으로 예측돼?
- 지금 온도·염도·pH 조건에서 품질 예측해줘
- 예측 신뢰도가 얼마야?
- 발효 완성도 예측에 어떤 데이터가 쓰였어?
- ML 예측이랑 실제 발효 상태가 맞아?
- 품질 등급 A가 되려면 어떤 조건을 맞춰야 해?

### 📊 대시보드 질문
- 대시보드에서 뭘 볼 수 있어?
- 지금 공정 상태 한눈에 보여줘
- 센서 이력 차트 어떻게 봐?
- 발효 진행률이 몇 %야?
- 오늘 온도 변화 그래프 보여줘
- 품질 예측 결과를 차트로 볼 수 있어?

### 🔧 임베딩 설정 질문
- 지금 어떤 임베딩 모델 쓰고 있어?
- 로컬 임베딩과 OpenAI 임베딩 중 어떤 걸 써?
- 임베딩 없이도 RAG가 동작해?
- 로컬 Ollama 모델로 전환할 수 있어?
- 검색 품질이 얼마나 돼?
